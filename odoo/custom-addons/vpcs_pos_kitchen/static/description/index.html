POS Kitchen

This module provides POS Kitchen Screen for Chefs.

Also adds order priority, limited categoriesed products in POS and product with different color in background according to POS Category in Kitchen Screen.

Features @Kitchen Screen

  • More attractive and user friendly screen.
  • Order Priority, Waiting Time, Order line notes etc. Visible in the screen.
  • Chef can update the state of order line to Cooking, Ready to Serve, Done (Served)
  • Easily identify product based on category color.
  • Kitchen Screen will make `Bell` sound when new order or new line ordered.
  • Bus Notification Concept.
  • Allow user to "Refresh" manually

Other Features

  • You can filter the screens by product categories. eg. Veg. Kitchen Screen will show only Veg. categoriesed products.
  • Waiter/Cashier can set the priority of the order.
  • Cachier/Waiter screens are also auto-synchronized with above states when it changed.
  • Configure color in POS Category.
