What Does Facebook Catalog Integration Module Do?

The module fastracks and automates the process of creating Facebook ad feeds of your Odoo products.

The module is designed to quickly produce data feed of the products containing details such as product information, prices, availability, etc, which can then be uploaded directly to the Facebook catalog manager with a click.

Generate feeds for your Odoo products in a few moments using Odoo Facebook Market module.

The module reduces the arduous and monotonous effort involved in manually creating the XML file for the products information. It automatically renders an XML data feed for the chosen product or products in Odoo for the user.

The User then just needs to add the file to the Facebook catalog manager to generate the Product feeds.

What Is Facebook Catalog manager

The Facebook catalog manager holds the information about your products, The information such as product information, prices, availability, etc. is stored in the Facebook Product catalog by the seller.

The information is picked by Facebook to show the most relevant ads to users on Facebook, Instagram and Messenger according to your selected Ads campaign.

Different catalogs can be created for different types of products.

For more information click on the following link >> https://www.facebook.com/business/help/890714097648074

Features:

  • The module simplifies the process of creating product data feeds for Facebook catalog.
  • The user can create the product Data Feed in the XML file format for Odoo products.
  • The Data feeds can be created for and individual product, product categories, and product domain.
  • The file can then be uploaded to the Facebook catalog manager.
  • Multiple catalogs can be created for the products as per the requirements.
  • Once uploaded, the catalog is saved in the Facebook catalog manager.
  • The user can also add the standard Google product Categories to the Odoo products.
  • The user can choose which product information fields to be included in the XML file.
  • It also provides the option to select the language and pricelist in which you want to upload the data feed on Facebook.
  • The data feeds can be created manually or using the Cron scheduler in Odoo.
  • You can map the Google product categories with the Odoo product categories in Odoo.
  • The Odoo product fields can also be mapped with those of Facebook product fields in Odoo backend for accurate data synchronization.

Prerequisites for Using Facebook Catalog Manager

  1. You need to create a Facebook Business account to manage your ads >> https://www.facebook.com/business/pages/set-up

  2. Next, you would also need a Facebook Ads Manager account >> https://www.facebook.com/business/tools/ads-manager

  3. Once you have set up the Facebook Ads Manager, you can head to Facebook Catalog manager to upload the product data to the product catalog. To know more click on the following link >> https://www.facebook.com/business/help/1754901084745017

Setting up the module

  1. Once you have set up your Facebook Business Console, create the Catalog
  2. Now, select the product to Add Product > Add New Products
  3. Then Select "Use Data Feeds" and click Next
  4. next, select "Set a Schedule" and select the time after that time the scheduler will automatically works
  5. Enter the URL created from the Catalos in the Addon
  6. Click the Create "Start Upload" button to start Uploading
  7. Then It will automatically start the scheduler
